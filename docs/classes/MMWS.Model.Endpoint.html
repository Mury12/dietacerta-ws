\MMWS\Model

\MMWS\ModelEndpoint

Creates endpoints and configure it.

All endpoints created will need to be set to any HTTP method GET|DELETE|POST|PATCH|PUT and will be used to set routes.

Example Usage:

use MMWS\Model\Endpoint;

return ['ws' => ['body' => $e = new Endpoint(), $e->get('domain/file', 'method')->permission('auth')]]
